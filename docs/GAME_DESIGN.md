# Frog Fishing Game

## Fantasy

Alien frog overcomes depression by feeding communities and becoming a part of the world.

## Pillars

- Connection
- Exploration
- Fishing
- Community Growth
- Cozy Atmosphere

## Core Loop

Explore -> Fish -> Feed -> Befriend -> Unlock -> Explore

## Features

### Fishing

Main activity.

Purpose:
- Feed communities.
- Unlock content.
- Build relationships.

### Companions

NPCs accompanying player.

Effects:
- Modify gameplay.
- Unlock interactions.

### Mining

Secondary exploration activity.

Purpose:
- Obtain materials.
- Find relics.
- Access underground fishing spots.
